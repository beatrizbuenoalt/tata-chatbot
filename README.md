# Táta Chatbot 🤖

A simple chatbot built with Python, made as a learning project while studying Python fundamentals.

## What it does

Táta introduces herself, asks for your name, age, and favorite color, and responds with a bit of personality depending on your answers. It only communicates in English, and blocks common Portuguese words as a small language-filter exercise.

## Concepts practiced

- Input validation with `while` loops
- Error handling with `try/except`
- Conditional logic (`if` / `elif` / `else`)
- String methods (`.lower()`, `.capitalize()`, `.isalpha()`, `.replace()`)
- Dictionaries to store user answers
- External libraries (`colorama` for colored terminal output)

## How to run

1. Clone this repository
2. Install dependencies: pip install -r requirements.txt
3. Run the bot: python bot.py
